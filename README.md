# Симулятор Казино

Симулятор казино с игроками и гусями, реализованный на Python.

## Описание

Игровой симулятор, где игроки делают ставки с фишками, а различные типы гусей влияют на ход игры через случайные события.

## Основные компоненты

### Классы игроков
- **Player** - базовый игрок с фишками и показателем паники
- **ChipCollection** - управление коллекцией фишек
- **Chip** - отдельная фишка с номиналом

### Классы гусей
- **Goose** - обычный гусь
- **WarGoose** - боевой гусь (атакует игроков)
- **HonkGoose** - кричащий гусь (повышает панику)
- **GooseFlock** - стая гусей

### Игровые механики
- **Ставки** - игроки делают ставки фишками
- **Атаки** - боевые гуси атакуют игроков
- **Крики** - кричащие гуси повышают панику
- **Объединения** - гуси объединяются в стаи

## Возможности

- Пошаговая или полная симуляция
- Сохранение и загрузка игр в JSON
- Поддержка seed для воспроизводимости
- Статистика игроков и гусей
- Система паники и выбывания игроков

## Правила игры

1. Игроки начинают с определенным балансом фишек
2. В каждом шаге происходит одно из событий:
   - Ставки игроков с случайным коэффициентом
   - Атака боевого гуся на случайного игрока
   - Крик гуся, повышающий панику игрока
   - Объединение гусей в стаю
3. Игрок выбывает при панике >92% или балансе <5 фишек
4. Побеждают игроки при достижении 5000 фишек или гуси при выбывании всех игроков
